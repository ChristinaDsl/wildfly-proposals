== Requirements

Replace all usage of `master`, `slave`, `blacklist` and `whitelist` both in resource addresses and UI elements:

* `master` → `primary`
* `slave` → `secondary`
* `blacklist` → `blocklist`
* `whitelist` → `allowlist`
